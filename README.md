# 🧠 Fake Job Detection System + Chrome Extension

## 📌 Overview

This project detects whether a job posting is real or fake using Machine Learning and NLP.
It also provides suggestions like **Apply / Avoid** and works as a Chrome Extension.

---

## 🚀 Features

* Fake vs Real job detection
* Smart suggestion system
* Chrome Extension support
* Flask API backend
* Streamlit Web App
* Selenium for job link extraction

---

## 🛠️ Technologies

* Python
* Scikit-learn
* NLP (TF-IDF)
* Flask
* Streamlit
* Selenium
* JavaScript (Chrome Extension)

---

## ⚙️ Setup

### 1. Install dependencies

pip install -r requirements.txt

---

### 2. Run API

python api.py

---

### 3. Run App

streamlit run app.py

---

### 4. Load Extension

* Open chrome://extensions/
* Enable Developer Mode
* Load job-detector-extension folder

---

## 🧪 Usage

* Open any job page
* Click extension
* Analyze job → get result
